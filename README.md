# Конвертер Mp3 в Opus

Это простой веб-инструмент для конвертации файлов из формата MP3 в Opus.

## Возможности

*   Загрузка файлов MP3.
*   Конвертация файлов MP3 в формат Opus с помощью FFmpeg.
*   Скачивание сконвертированного файла Opus.

## Требования

*   Python 3
*   Flask
*   FFmpeg

## Как использовать

1.  Клонируйте репозиторий:
    ```
    git clone https://example.com/your-repo.git
    ```
2.  Установите необходимые пакеты Python:
    ```
    pip install Flask
    ```
3.  Установите FFmpeg. Если вы используете Mac, вы можете воспользоваться Homebrew:
    ```
    brew install ffmpeg
    ```
    Для других операционных систем, пожалуйста, обратитесь к официальному сайту FFmpeg: [https://ffmpeg.org/](https://ffmpeg.org/)

4.  Запустите приложение:
    ```
    python main.py
    ```
5.  Откройте ваш веб-браузер и перейдите по адресу `http://127.0.0.1:5000/`.

6.  Загрузите MP3-файл и дождитесь окончания конвертации.

7.  Нажмите кнопку "Скачать", чтобы загрузить сконвертированный файл Opus.

## Как это работает

Приложение создано с помощью Flask, легковесного веб-фреймворка для Python.

Когда вы загружаете MP3-файл, он сохраняется в директорию `uploads`. Затем приложение использует FFmpeg для конвертации MP3-файла в формат Opus. Сконвертированный файл сохраняется как `out.opus` в директории `uploads`.

Наконец, приложение предоставляет ссылку для скачивания сконвертированного файла.

## Файлы

*   `main.py`: Основной файл приложения.
*   `templates/upload.html`: Страница загрузки файла.
*   `templates/download.html`: Страница скачивания файла.
*   `uploads/`: Директория, в которой хранятся загруженные и сконвертированные файлы.
